# Rustic Garden — Sustainable Agroforestry & Web Architecture

An open-source, client-side digital portal for an architectural farm project located in Kavutiri, Embu, Kenya. This platform integrates agroecological land management metrics with a minimal graphic design approach.

## 🛠️ Design & Hydrology Parameters
- **Zero Cost Architecture:** Built entirely with static HTML5 and native CSS Custom Variables. It uses dynamic URL routing for invoicing without backend hosting requirements.
- **Visual Token System:** Typography centers around contrasting layouts featuring *Fraunces* and *Space Mono* built over an organic color palette inspired by soil chemistry and coffee cherry profiles.
- **Hydrological Integration:** Features integrated layouts communicating multi-tier rainwater tracking alongside regenerative smallholder coffee loops.
